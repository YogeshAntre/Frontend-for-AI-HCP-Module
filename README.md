- Topics Discussed
- Materials Shared
- Samples Distributed
- HCP Sentiment
- Outcomes
- Follow-up Actions

# 3. Redux State Management

Redux Toolkit is used for global state management.

The interaction data is stored and updated using Redux.


Implemented Actions:

### updateInteraction

Updates interaction details in the UI.

### clearInteraction

Clears interaction state.
